## PyCharm 使用技巧记录

> ### 快捷键
#### 自动导包设置
* 在PyCHarm >Preferences >auto import 勾选show import popup
* 在 keymap >main Menu> Code >Completion>Basic 中设置 导包快捷键
* 在 file–>setting–>project:Python–>project structure 把需要导入的module(自己的) 所在目录设置为source
* 本人设置为 `shift` + `1`
#### 自动生成类中不存在的函数
* 在使用的函数处 `alt` + `enter`
